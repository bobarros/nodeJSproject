## Getting Started

### 1. Clone this Repository

```
git clone https://github.com/bobarros/nodeJSproject.git

```

### 2. Navigate to the directory created

```
cd nodeJSproject
```

### 3. Install dependencies

```
npm install
```

### 4. Run Eleventy

```
npm run startlocal
```
